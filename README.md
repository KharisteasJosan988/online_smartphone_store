# Online Smartphone Store (Semicolon)

A web-based e-commerce application for selling smartphones, built using Laravel.  
This project supports product management, shopping cart, checkout, and order processing.

---

## Features

### User Features
- User registration & login
- View product list
- Filter products by brand
- Add to cart
- Checkout system
- Select courier & shipping cost (RajaOngkir API)
- Order history

### Admin Features
- Dashboard management
- CRUD products
- CRUD categories
- Manage couriers
- Manage orders
- Update order status (pending, processed, shipped, delivered)
